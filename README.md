SummaryTables lets you create summary tables and descriptive statistics
directly in jamovi — Table 1 with group comparisons, cross tabulations,
Likert-scale summaries, and Word export. Built on the
[gtsummary](https://www.danieldsjoberg.com/gtsummary/) R package.

## Features

- **Descriptive statistics** — mean, SD, median, IQR, range for
  continuous variables; counts and percentages for categorical variables
- **Group comparisons** — split tables by a grouping variable (Table 1 style)
- **Cross tabulation** — cross-tabulate two categorical variables with
  counts, percentages, and row/column margins
- **Likert-scale summaries** — response distributions for ordered
  categorical items
- **Statistical tests** — t-test, ANOVA, Wilcoxon rank-sum, Kruskal-Wallis,
  Chi-square, Fisher's exact test, Z test for proportions
- **Effect sizes** — Cohen's d, Hedge's g, standardized mean difference (SMD)
- **Confidence intervals** — for means and proportions (Wilson,
  Clopper-Pearson, Wald, Agresti-Coull, Jeffreys)
- **P values** — with adjustment methods (Benjamini-Hochberg, Bonferroni,
  Holm, and others)
- **Word export** — save tables directly as .docx files
- **Journal formatting** — JAMA, The Lancet, NEJM, Quarterly Journal
  of Economics
- **16 languages** — English, French, German, Spanish, Chinese, Japanese,
  Korean, and more
- **Per-variable control** — override statistics, tests, and methods for
  individual variables
- **Missing data handling** — show, hide, or always display with
  customizable labels
