# SummaryTables

A [jamovi](https://www.jamovi.org) module for publication-ready summary and regression tables, powered by [gtsummary](https://www.danieldsjoberg.com/gtsummary/).

## Overview

SummaryTables is designed to make creating "Table 1" summaries and regression output as effortless as possible directly from the jamovi interface. Whether you are preparing a manuscript for a major journal or need a quick descriptive breakdown of your data, SummaryTables automates the statistical testing, effect sizes, and formatting.

## Features

- **Word export:** Save any table directly as a `.docx` file 
  *(Note: Direct Word export is only available in the desktop version of Jamovi, not the cloud version for security reasons).*
- **Journal formatting:** Built-in themes for major journals including JAMA, The Lancet, NEJM, and Quarterly Journal of Economics.
- **16 languages:** Output tables in English, French, German, Spanish, Chinese, Japanese, Korean, and more.

## Included Analyses

- **Summary Table:** Create classic "Table 1" summaries with group comparisons, automatic statistical testing, effect sizes, and p-value adjustments.
- **Continuous Table:** Summarize a single continuous variable across multiple categorical variables with automatic statistical testing and p-value adjustments.
- **Cross Table:** Summarize the association between two categorical variables using cross-tabulations with automatic statistical testing.
- **Likert Table:** Summarize Likert scale items easily.
- **Survival Table:** Create Kaplan-Meier survival tables showing survival probabilities and median survival times.
- **Multivariable Regression:** Fit multivariable linear, logistic, or Cox regression models.
- **Univariable Regression:** Run separate univariable linear, logistic, or Cox regression models for each predictor.

### Installation
You can install SummaryTables directly from the jamovi library! Open jamovi, click on the **Modules** icon (the plus sign in the top right), select **jamovi library**, and search for "SummaryTables".
